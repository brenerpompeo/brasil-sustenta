| Componente                | Valor              |
|---------------------------|--------------------|
| **Ticket mínimo**         | R$ 15.000          |
| **Ticket máximo**         | R$ 25.000          |
| **Duração**               | 6–8 semanas        |
| **Talentos no squad**     | 3–5                |
| **Entregável**            | Relatório de impacto + artefatos do sprint |
| **Custo direto estimado** | R$ 7.000 – R$ 10.000 |
| **Margem bruta**          | ~40%               |

**O que inclui:**
- [ ] Brief estruturado pelo time BS
- [ ] Shortlist com ODS Fit Score explicado (top 5)
- [ ] Kick-off e 2 checkpoints de sprint
- [ ] Relatório final com trilha de evidência

**O que não inclui:** Contratação, NDA corporativo extendido, integração de sistemas

## Notas Táticas

- **Preços são estimativas** — ajustar após 1º pilot fechado com feedback real do comprador
- **Não colocar preços no site** até ter pelo menos 2 cases fechados (exceto faixas orientativas)
- **Desconto máximo recomendado:** 15% para 1ª compra / early adopter
- **Faturamento:** Pessoa Jurídica (CNPJ) — verificar CNAE e regime tributário antes de emitir NF
- **Prazo de pagamento:** 50% no kick-off + 50% na entrega final (Pilot) · Mensalidade (Managed)
